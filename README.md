[ 软件简介 ]
本工具能够将五线谱截图自动识别为 MIDI 音乐，并立即播放。
无需安装 Python 或任何运行环境，下载 exe 文件后双击即可使用。

[ 主要功能 ]
- 自定义快捷键区域截图，自动保存
- 自动识别五线谱，生成 MusicXML 和 MIDI 文件
- 一键播放识别出的旋律，支持自动播放模式
- 三标签页管理：截图、XML 文件、MIDI 文件
- 支持切换音色库（.sf2 文件），调节音量与播放速度
- 音量、速度、音色库和快捷键设置自动保存

[ 使用方法 ]
1. 打开“小花截图MIDI播放工具.exe”。
2. 通过菜单或工具栏进行截图，或使用默认快捷键 Ctrl+Shift+Z 框选乐谱区域。
3. 截图后，在缩略图上右键选择“识别并播放”，也可勾选工具栏的“截图后自动播放”。
4. 在工具栏可调节音量、播放速度，以及切换不同的音色库。
5. 生成的 XML 和 MIDI 文件可在对应的标签页管理，支持打开、转换、播放与删除。

[ 文件夹说明 ]
程序会在 exe 所在目录自动生成以下文件夹，用于存放个人文件：
- screenshots   : 存放你的乐谱截图
- xml_outputs   : 存放识别生成的 MusicXML 文件
- midi_outputs  : 存放转换后的 MIDI 文件
- soundfonts    : 存放音色库文件（可放入多个 .sf2 自由切换）

[ 更换音色库 ]
你可以将下载的 .sf2 音色库文件放入“soundfonts”文件夹中。
启动程序后，在工具栏的“音色”下拉框中选择你想要的音色库即可。
推荐音色库：FluidR3_GM.sf2、Timbres Of Heaven.sf2 等。

[ 常见问题 ]
问：播放没有声音？
答：请检查系统默认音频设备，确保音量合成器中本程序未被静音。
    同时确认 soundfonts 文件夹内有有效的 .sf2 文件，并在工具栏中已选中。

问：快捷键无效？
答：可在菜单“设置”中修改为其他组合键，避免与其他软件冲突。

问：播放速度如何调整？
答：拖动工具栏上的“速度”滑块即可改变播放速度，不会影响音高。
